## 📋 الأوامر الكاملة

### للجميع:
- `/start` — الصفحة الرئيسية
- `/help` — قايمة الأوامر
- `/ping` — اختبار البوت
- `/info` — معلومات البوت

### 👑 المالك فقط:
- `/publish` — **نشر منشور** (القلب بتاع البوت!)
- `/addchannel @username` — أضف قناة
- `/removechannel [id]` — شيل قناة
- `/listchannels` — شوف كل القنوات
- `/addbot @username` — أضف بوت/مجموعة
- `/removebot [id]` — شيل بوت
- `/listbots` — شوف كل البوتات
- `/report [id]` — تقرير قناة
- `/ban [id]` — حظر مستخدم
- `/unban [id]` — رفع حظر
- `/cancel` — إلغاء العملية الحالية

## 📤 إزاي تنشر منشور؟

1. ابعت `/publish`
2. **اختار الوجهة:** قناة معينة أو كل القنوات أو كل البوتات أو الكل
3. **اختار النوع:** نص / صورة / فيديو / ملف / استطلاع / تحويل
4. **ابعت المحتوى**
5. **راجع وأكد** → البوت بينشر على طول!
